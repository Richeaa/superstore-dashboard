# Power BI Sales Dashboard

---

## Project Overview

This dashboard provides a comprehensive analysis of sales data using Microsoft Power BI. The report is designed to help users:

* Monitor overall sales performance
* Analyze sales by state and region
* Explore category and segment insights
* Identify top-performing products
* Track KPIs and business metrics visually

---

## Dashboard Pages

### 1. Dashboard

Provides a high-level overview of:

* Total Sales
* Profit
* Quantity Sold
* Key KPIs
* Sales trends and performance summaries

### 2. State Analysis

Focuses on geographical insights:

* Sales distribution by state
* Regional performance comparison
* Location-based analysis using maps

### 3. Category & Segment Analysis

Analyzes:

* Product categories
* Customer segments
* Profitability by category
* Segment contribution to revenue

### 4. Products Analysis

Displays:

* Top-selling products
* Product profitability
* Product performance comparison
* Product trends
